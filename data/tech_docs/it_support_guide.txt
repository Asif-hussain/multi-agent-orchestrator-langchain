ADVANCED TROUBLESHOOTING

Browser Issues and Performance

Google Chrome Problems:
- Browser running slow: Clear cache and cookies (Settings > Privacy > Clear browsing data)
- Extensions not working: Disable and re-enable, or reinstall from Chrome Web Store
- Pages not loading: Clear DNS cache, disable extensions, try incognito mode
- Crashes frequently: Update to latest version, disable hardware acceleration
- Memory usage high: Use Chrome Task Manager (Shift+Esc) to identify heavy tabs

Firefox Issues:
- Sync not working: Sign out and sign back into Firefox account
- Add-ons causing problems: Start in Safe Mode (Help > Restart with Add-ons Disabled)
- Slow performance: Refresh Firefox (About Firefox > Refresh)
- Certificate errors: Check system date/time settings

Microsoft Edge Issues:
- Pages displaying incorrectly: Clear cache, disable Compatibility Mode
- Collections not syncing: Check Microsoft account sync settings
- PDF viewer not working: Reset Edge settings or use Adobe Reader
- Favorites missing: Sign into Microsoft account to restore

Browser Best Practices:
- Keep browser updated to latest version
- Use maximum 10-15 tabs simultaneously for optimal performance
- Clear cache monthly
- Disable unused extensions
- Use bookmarks instead of keeping tabs open
- Enable pop-up blocker
- Set homepage to company portal

Disk Cleanup and Maintenance:

IT SECURITY BEST PRACTICES

Password Management:

Password Requirements:
- Minimum 12 characters (16+ recommended)
- Mix of uppercase and lowercase letters
- Include numbers and special characters
- No dictionary words or personal information
- Cannot reuse last 10 passwords
- Must change every 90 days
- Cannot contain username or email

Using Password Managers:
- 1Password or LastPass recommended
- Store all passwords securely
- Generate strong random passwords
- Sync across devices
- Share credentials securely with team members
- Enable biometric authentication
- Use master password that you'll never forget

Password Security Tips:
- Never write passwords down
- Don't share passwords via email or Slack
- Use different password for each system
- Enable password manager autofill
- Be cautious of shoulder surfing
- Log out of shared computers
- Use passphrases for memorable strong passwords

Encryption and Data Protection:

BitLocker (Windows):
- Full disk encryption enabled on all company laptops
- Encrypts entire hard drive
- Protects data if device lost or stolen
- Transparent to user (no performance impact)
- Recovery key stored with IT

FileVault (Mac):
- Full disk encryption for macOS
- Enabled by default on company Macs
- Encrypts all data at rest
- Uses login password to unlock
- Recovery key stored in iCloud and with IT

Data Classification:
- Public: Marketing materials, press releases
- Internal: General business information, policies
- Confidential: Employee data, contracts, financial info
- Restricted: Trade secrets, unreleased products, PII

Handling Confidential Data:
- Store only on company-approved systems
- Encrypt when emailing externally
- Shred physical documents
- Lock computer when away from desk
- Use privacy screens in public places
- Report data breaches immediately

USB and External Drive Policy:
- Use only company-approved USB drives
- Encrypt sensitive data on external drives
- Scan all USB drives for malware before use
- Do not use personal USB drives for company data
- Return company USB drives when no longer needed

How to Submit Effective IT Tickets:

Include in Your Request:
- Detailed description of the issue
- What you were doing when it occurred
- Error messages (take screenshot)
- Steps you've already tried
- Impact on your work (critical, high, medium, low)

"Unable to print to 3rd floor color printer. Error message: 'Printer offline.' Tried restarting printer and removing print jobs from queue. This is blocking urgent client proposal due today at 4 PM. Please call me at ext. 5555. Thank you!"

"Printer broken. Fix it."

- Updates sent via email
- Respond to IT requests for information
- Close ticket when issue resolved
- Provide feedback on service quality

Escalation Process:
If issue not resolved in expected timeframe:
1. Reply to ticket with urgency explanation
2. Request supervisor review
3. Contact IT Service Desk Manager
4. Email CIO for critical business-impacting issues
